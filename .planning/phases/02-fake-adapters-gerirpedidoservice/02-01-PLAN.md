# Plano 02-01 — Fake Adapters

## Objectivo
Implementar os adaptadores simulados (Fakes) em memória para os Ports de suporte desenvolvidos na Fase 1. Estes Fakes permitirão testar o Domain e os Services do POS sem a necessidade de uma base de dados ou da infraestrutura do Laravel. Os fakes serão alojados nos próprios módulos, acompanhados por Seeders para injectar dados de teste.

## Contexto
Na Fase 1, foram criados os Ports:
- Módulo Stock: `BuscadorDeProdutoPort`, `VerificadorDeStockPort`, `StockRepositorioPort`
- Módulo Clientes: `ClienteRepositorioPort`
- Módulo Pagamentos: `PagamentoRepositorioPort`

Nesta tarefa, serão criadas classes em PHP puro nas directorias `app/Modules/{Modulo}/Adapters/Saida/Fakes/` que implementam estas interfaces usando arrays ou variáveis estáticas em memória.
