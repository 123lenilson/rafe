# Roadmap

## Phase 1: Dashboard Layout e Estrutura Base
**Goal:** Estabelecer a estrutura fundamental da interface (Rotas, Sidebar, TopBar)
**Requirements:** CORE-01, CORE-02, NAV-01, NAV-02, TOP-01
**Status:** Concluída
**Success criteria:**
1. A aplicação carrega com um layout que contém uma Sidebar à esquerda e uma TopBar.
2. A Sidebar tem um design preto e branco, minimalista (Square-style).
3. A Sidebar suporta submenus deslizantes para navegação.
4. O React Router está configurado e permite a renderização de rotas filhas dentro do DashboardLayout.
